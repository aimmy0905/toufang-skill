# Google Ads Brief Agent

## 这是什么？

这是整套 Google Ads Skills 的起点。

它负责把业务方提供的活动信息整理成一个标准化投放 Brief，明确：

- 要卖什么
- 卖给谁
- 目标是什么
- 时间和预算是什么
- 成功标准是什么
- 当前还缺哪些关键信息

## 它解决什么问题？

很多投放问题并不是出在广告设置本身，而是出在一开始就没有把目标、约束和 KPI 讲清楚。

这个 Skill 的任务是先把业务需求翻译成投放可用的输入。

## 输入

- 活动名称
- 主题
- 产品信息
- 市场与语言
- 时间范围
- 预算
- 投放目标
- 转化动作
- KPI 目标
- 业务约束
- 历史经验

## 输出

- Campaign Brief
- Goal Summary
- Priorities
- Risks
- Missing Information

## 它不做什么？

- 不拉平台数据
- 不生成最终广告文案
- 不直接做后台搭建
