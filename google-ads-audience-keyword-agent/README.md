# Google Ads Audience Keyword Agent

## 这是什么？

这个 Skill 负责把投放 Brief 进一步拆成可执行的人群策略和关键词策略。

它重点解决：

- 人群怎么分层
- 用户搜索意图怎么分类
- 关键词该怎么组织
- 匹配方式怎么选
- 否定关键词应该拦什么

## 输出重点

- Audience Segments
- Search Intent Map
- Keyword Themes
- Match Type Guidance
- Negative Keyword Suggestions

## 它不做什么？

- 不直接创建广告
- 不直接读取搜索词报表
- 不负责生成最终诊断结果
