# Google Ads Setup Agent

## 这是什么？

这个 Skill 把策略翻译成后台可执行的搭建方案。

它负责定义：

- Campaign 怎么拆
- Ad Group / Asset Group 怎么拆
- 预算怎么分
- 出价怎么选
- 命名规则怎么统一
- 最小追踪清单有哪些

## 输出重点

- Campaign Structure
- Naming Rules
- Budget Plan
- Bidding Guidance
- Tracking Checklist

## 它不做什么？

- 不直接调用 Google Ads API 建广告
- 不验证追踪代码是否真的部署成功
